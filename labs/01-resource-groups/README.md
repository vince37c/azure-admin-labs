# Lab 01: Create a Resource Group

## Objective
Create a Resource Group to organize Azure resources by workload and environment.

## Resources Created
- 1 Azure Resource Group

## Prerequisites
- Azure subscription access
- Contributor or Owner permissions

## Build Steps (Azure Portal)

1. Sign in to Azure Portal.
2. Search for Resource groups.
3. Select Create.
4. Enter:

Resource Group Name:
rg-adminlabs-dev-eastus

Region:
East US

5. Select Review + Create
6. Select Create

## Validation
- Confirm resource group appears in Resource Groups list.
- Open the Resource Group Overview page.

Screenshot required:
- Resource Group Overview

## Cleanup
1. Open the Resource Group.
2. Select Delete resource group.
3. Type the resource group name to confirm.
